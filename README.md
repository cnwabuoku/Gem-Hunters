# Gem Hunters

A game console application in C#

The classes of this application are placed in separate C# files

Steps:

Clone repository in Visual studio

Run the game application

Choose L,R,U,D to move a player Left, Right, Up, and Down respectively

Gems are represented by "G", Obstacles are shown as "O"

The player that collects the most gems wins the game.
